Название программы: Программа учета сбора картофеля

Описание программы:
Программа разработана для учета производственной деятельности по сбору картофеля. Каждая бригада имеет свой номер и лидера, а также ведет учет количества собранного картофеля и даты сбора урожая. Программа позволяет определить количество собранного картофеля для каждой бригады, фамилию лидера в бригаде, выдает текущий результат после любых изменений и вычисляет суммарный результат по всем бригадам. Также программа позволяет определить самый урожайный год.

Функциональные возможности программы:
- Ввод данных о количестве собранного картофеля для каждой бригады;
- Ввод данных о фамилии лидера в бригаде;
- Ввод данных о дате сбора урожая;
- Оперативный расчет текущего результата после любых изменений;
- Вычисление суммарного результата по всем бригадам;
- Определение самого урожайного года.
